# Temple Run game

## Design

- blocky pattern similar to something like lego

- lego player with animations

- 3 dimentional

- vision shifts for a different experiance and variation

- many obstacles 

- rubber band sphere/ball chasing player

- one hit death

## Obstacles

- Fire

- pit

- wall that falls

- trip wire

- turn

- part of walkway falls
